# Bootstrap 5 Exercises

This folder contains completed exercises for the Bootstrap 5 assessment. Open the HTML files in a browser to view examples.

Files:
- `index-cdn.html` – All exercises demonstrated using the Bootstrap CDN (recommended to preview).
- `index-local.html` – Same examples but using the provided `vendor/bootstrap` files (downloaded setup demo).
- `structure.md` – Explanation of Bootstrap directory structure.
- `package.json` & `scss/` – Example npm/Sass setup for customizing Bootstrap.

How to preview:
1. Open `index-cdn.html` in your browser.
2. (Optional) For the local example, open `index-local.html`—the vendor files are placeholders; to use real Bootstrap, replace `vendor/bootstrap` with actual download or install via npm and build.

To compile Sass (example):

```
npm install
npm run build-css
```
